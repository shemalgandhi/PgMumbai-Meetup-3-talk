# PgMumbai-Meetup-3-talk
This is the slide deck which I presented at PgMumbai meetup #3 on June 19th 2026.

Topic: "Beyond Async I/O : Postgres 18's Hidden Gems"

While Async I/O rightfully steals the spotlight as a game-changing feature in Pg18, there's a lot more under the hood that doesn't get the attention it deserves. 
This talk is my small effort to surface  some of these lesser-known features and enhancements, and help the community discover ways to put them to good use.
